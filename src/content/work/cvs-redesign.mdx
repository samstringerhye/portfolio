# CVS Shop Website Redesign

CVS.com is a major e-commerce platform serving millions of customers. The shop had grown inconsistent. Visual hierarchy was unclear, navigation was fractured, and the system hadn't kept pace with the brand. The modernization brought clarity back to a platform that had gotten complicated.

## Approach

I focused on the foundations: typographic framework, icon system, and reusable components built to scale across templates. When the system is right, everything downstream gets better.

### Typography & Icons

I designed a custom icon and illustration system for CVS using their brand red. Each icon was built on a grid to keep proportions consistent at any size.
